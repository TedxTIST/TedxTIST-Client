Place Font Awesome CSS and webfonts here. Download from https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.0/ or https://fontawesome.com/download.

Required files:
- all.min.css
- webfonts/* (all font files)

After placing files, import the CSS in your globals.css or layout.tsx:
@import "/fontawesome/all.min.css";

This enables CSP-safe local Font Awesome usage.
